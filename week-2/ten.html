1. HTML

    Primary Use Cases Key Features/Benefits
    Building the structure of web pages
    1. Simple and easy to learn
    2. Compatible with web browsers
    3. Allows for semantic markup

  2. JS

    Primary Use Cases Key Features/Benefits
    To make web pages interactive
    1. Scripting Language made for client-side execution on the browser
    2. Interpreted language instead of a compiled one
    3. Enables you to handle events and even generate custom events.

  3. CSS

    Primary Use Cases Key Features/Benefits
    Used to style and layout web pages
    1. CSS allows for the separation of the content structure (HTML) from its visual presentation (CSS).
    2. With CSS, you can define styles once and apply them to multiple elements throughout your website.
    3. CSS provides a wide range of selectors and properties that give you precise control over the appearance of your web pages.
