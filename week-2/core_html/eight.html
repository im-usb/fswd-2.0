HTML
A markup language used for creating the structure and content of web pages.
CSS
A style sheet language used for describing the presentation and appearance of a web page.
JavaScript
A programming language used for adding interactivity and dynamic behavior to web pages.
Node.js
A runtime environment that allows JavaScript to be executed on the server-side, enabling server-side scripting and building scalable network applications.
Express.js
A minimal and flexible web application framework for Node.js that simplifies the process of building server-side applications and APIs.
React.js
A JavaScript library for building user interfaces. It allows developers to create reusable UI components and efficiently update the user interface in response to changes in data.
MongoDB
A NoSQL document-oriented database that provides flexibility and scalability for handling unstructured or semi-structured data.
